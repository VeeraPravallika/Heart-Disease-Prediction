# Heart Disease Prediction Using Machine Learning

## Overview

This project predicts the likelihood of heart disease using machine learning techniques. The model analyzes important health parameters such as age, blood pressure, cholesterol level, chest pain type, and heart rate to estimate the risk of heart disease.

## Features

* Data preprocessing and feature scaling
* Random Forest Classifier
* Accuracy, Precision, Recall and F1 Score evaluation
* Confusion Matrix and Classification Report
* Interactive prediction system
* Risk level assessment

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn

## Dataset Features

* Age
* Gender
* Chest Pain Type
* Blood Pressure
* Cholesterol
* Maximum Heart Rate
* Exercise-Induced Angina

## Model Performance

* Accuracy: 98.54%
* Precision: 100%
* Recall: 97.09%
* F1 Score: 98.52%

## How to Run

Install dependencies:

pip install -r requirements.txt

Run the project:

python heart_disease_prediction.py
